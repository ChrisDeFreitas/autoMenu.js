AutoMenus Help Doc

Purpose
- dynamically load HTML for menus from an external HTML file
- allow menus to be edited simply and have changes applied to existing HTML docs
- speed generation of HTML documents by reducing redundant code


Requirements
1. expect autoMenu home folder off webserver root:
	http://connexions.org/autoMenu/

2. expect folder to contain HTML menu templates:
	http://connexions.org/autoMenu/Templates

3. customized autoMenu.js to load required HTML templates, example:
	http://connexions.org/autoMenu/cxAutoMenu.js


Procedure
1. Insert into <head> section:
	<script src="/autoMenu/cxAutoMenu.js"></script> <!-- contains menuLoad() -->

2. Modify <body> tag:
	<body onload="menuLoad()">

3. Insert <div> with custom ID attribute in <body> section:
	<div id="T_CxTopMenu">AutoMenu: insert T_TopMenu.html here</div>

4. Result
AutoMenu: insert T_TopMenu.html here


Notes
- use this document as an example implementation
- menu templates are are standard HTML files, so they can be edited/copied/... easily
- CSS can be included in menu templates between <div> tags
- Javscript in menu templates will not work--use an external .js file.  This appears to be a browser security feature.
- example menu template: http://connexions.org/autoMenu/Templates/T_CxTopMenu.html


Fin!
